1. maven 설치
 - 자바 빌드 도구 => 빌드 자동화
 - 빌드 : 프로젝트 생성 => 라이브러리 설정 => 코드 작업 => 컴파일 => 테스트 => 패키지 생성 => 배포
 - 패키지 : 압축파일(jar, war)
 - mvn compile 
 - mvn package  => 이런식으로 mvn을 불러서 침
환경변수설정 => mvn을 찾기 편하게 mvn이 어디있는지 적어둠.
